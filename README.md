# CBU-X FRAMEWORK
Commodity-Backed Unit / Monnaie de Compensation Multilatérale

## 📖 Présentation
CBU-X est un framework théorique et opérationnel pour la construction d'un système monétaire alternatif, sans dette, sans intérêt, adossé à des biens réels.

## 🚀 Fonctionnalités principales
- Calculateur de taux X → euro
- API REST
- Smart Contract Solidity
- TVP-VAR
- Géométrie hyperbolique
- Simulation quasi-Monte Carlo
- Tableau de bord Streamlit
